Politique de Confidentialité

Dernière mise à jour : 24 janvier 2026

Introduction

Stackly est une extension de navigateur qui vous aide à organiser vos onglets par workspaces. Nous prenons votre vie privée très au sérieux.

Données collectées

Ce que nous collectons

Aucune donnée personnelle n'est collectée ni transmise à des serveurs externes.

Stackly stocke uniquement les données suivantes localement sur votre appareil :

  • Les noms et configurations de vos workspaces
  • Les URLs et titres des onglets sauvegardés dans vos workspaces
  • Vos préférences (thème, paramètres de suspension d'onglets)
  • Statistiques d'utilisation locales (nombre de switches, etc.)

Ce que nous ne collectons PAS

Données d'identification personnelle
Historique de navigation
Données de formulaire ou mots de passe
Données transmises à des tiers
Cookies de tracking
Analyse comportementale

Permissions utilisées

Permission Utilisation
tabs Lire et gérer les onglets pour les sauvegarder dans les workspaces
storage Sauvegarder vos workspaces et préférences localement
alarms Déclencher la sauvegarde automatique et la suspension d'onglets

Stockage des données

Toutes les données sont stockées via l'API chrome.storage.local (ou équivalent Firefox/Safari) :

  • Les données restent uniquement sur votre appareil
  • Les données ne sont jamais synchronisées avec des serveurs cloud
  • Les données sont supprimées si vous désinstallez l'extension

Partage des données

Nous ne partageons aucune donnée avec des tiers.

Stackly fonctionne entièrement en local, sans serveur backend ni analytics.

Sécurité

  • Aucune communication réseau n'est effectuée par l'extension
  • Le code source est disponible sur GitHub pour audit

Vos droits

Vous pouvez à tout moment :

  • Exporter vos données (via la fonction d'export de l'extension)
  • Supprimer toutes vos données en désinstallant l'extension
  • Effacer les données via les paramètres de l'extension

Contact

Pour toute question concernant cette politique de confidentialité :

  • GitHub : github.com/ultraviolettes/workspace-manager/issues

Modifications

Cette politique peut être mise à jour. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.

Stackly respecte votre vie privée. Vos données restent les vôtres.
